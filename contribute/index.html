Contribute to Donkey

Donkey is an opensource project to help accelerate the developement of self driving autos. See below how you can contribute.

Add a part.

Submit a github issue when you find a problem with the code.

Fix a bug or reply to issues.

  1. Submit a github issue that describes a feature that you want. This is where the conversation can start.
  2. Fork the code, make your change and submit a pull request.

Tests

Make sure all tests pass before submitting your pull request.

Run tests with: python -m unittest
